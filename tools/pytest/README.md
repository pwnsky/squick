# Pytest

## Pytest的优势

Pytest的优点如下：

- Pytest可以并行运行多个测试，从而减少测试套件的执行时间。
- 如果没有明确提到，Pytest有自己的方式来自动检测测试文件和测试功能。
- Pytest允许我们在执行过程中跳过测试的一个子集。
- Pytest允许我们运行整个测试套件的一个子集。
- Pytest是免费和开源的。
- 由于其简单的语法，pytest非常容易上手。



## 安装

```
pip3 install pytest
```



## 测试

```
pytest case
```
